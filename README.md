Digital-Sanctuary
A Digital Journal Web Application where users can securely register, log in, and manage their personal journals across devices. Built to provide a safe, accessible, and user‑friendly journaling experience.

Project Name: Digital-Sanctuary (Digital Journal App)
Project Scope:  
A responsive web application enabling users to create, edit, search, and delete journal entries. 
Authentication is handled via registered email and password, ensuring secure access from any device.

Features
User Registration with email and password
Secure Login for authenticated access
Journal Management: Create, edit, delete, and search entries
Cross‑Device Access using registered credentials
Responsive Design for seamless use on desktop and mobile

Usage
Register with a valid email and password.
Log in to access your personal journal.
Add new entries, search existing ones, edit content, or delete journals.
Access your journals securely from any device.

Conclusion
Digital-Sanctuary provides a secure, accessible, and user‑friendly platform for journaling. 
With authentication and CRUD functionality, it ensures that users can manage their thoughts and reflections anytime, anywhere.
